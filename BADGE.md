---

## 1. Purpose

This document defines the mandatory requirements for the **Istockhomes Verified** badge.

The badge represents:
- verified identity
- verified participation
- verified reputation
- verified access to the Istockhomes platform

The badge is not decorative.  
It is a **live trust indicator**.

---

## 2. Who This Applies To

These requirements apply to **all client applications** connected to Istockhomes, including:

- iOS / Android applications
- Web portals
- White-labeled or branded storefronts
- Viewer-only apps
- Developer-modified or resold applications

If an application connects to Istockhomes, this document applies.

---

## 3. Badge Requirement

All applications **must display** one of the following:

- “Verified by Istockhomes”
- The official Istockhomes Verified logo

The badge must be:

- Clearly visible
- Permanently present during normal use
- Not hidden behind menus or gestures
- Not visually misleading or minimized to obscurity

---

## 4. Placement

Acceptable placements include:

- Footer (recommended)
- Persistent header
- Fixed overlay element

Unacceptable placements include:

- Hidden screens
- Scroll-only visibility
- Splash screens only
- Temporary modals
- Disabled or non-interactive elements

The badge must be visible **at all times during participation**.

---

The badge **must be clickable**.

On interaction, it must resolve to a live verification endpoint hosted by Istockhomes.
